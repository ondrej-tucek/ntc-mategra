## Mategra

* source codes of project Mategra
* made by SvelteKit


### Install & Update
```bash
npm i             # install packages or
npm run reinstall # include also remove node_modules
npm run pkg:info  # info about packages
npm run pkg:up    # update packages
```

### Dev
```bash
npm run dev
```

### Pre-Deploy
```bash
npm run build
npm run preview
```

### Deploy to gh-pages
```bash
npm run build:gh
npm run deploy
```
